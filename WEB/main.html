Разработать ПО погодной станции.

Система должна состоять из двух модулей:
  1. Основной модуль:
    1. Имеет Web-интерфейс управления
    2. Отдает информацию по JSON, взаимодействует с соцсетями (по выбору)
    3. Синхронизирует время по NTP
    4. Отправляет команды/получает данные вспомогательному модулю на частоте 2.4GHz
  2. Датчик – измеряет температуру (и возможно другие параметры), передает информацию основному модулю.
  3. Платформа: Основной модуль: .Net micro, датчик: NXP ARM low-power
  4. Сложность: 3
